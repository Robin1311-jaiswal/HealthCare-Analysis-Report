# 1. Project Title / Headline
# Healthcare Patient Flow Intelligence Dashboard – Power BI | MySQL | Python | DAX
Multi-page Power BI dashboard providing insights into patient flow, admission trends, wait time analysis, demographic patterns, and hospital operational performance.
# 2. Short Description / Purpose

This interactive Power BI dashboard analyzes over 9,216 patient visits and 4,612 admissions to help healthcare stakeholders monitor patient demand, evaluate admission patterns, optimize wait times, and improve operational efficiency.
The dashboard provides a centralized analytics solution for understanding patient demographics, referral trends, department performance, and resource utilization.

# (iv) Business Impact & Insights
The hospital managed over 9,216 patient visits with a 50.04% admission rate.
General Practice emerged as the highest referral department.
Neurology recorded the longest average patient wait time (36.8 minutes).
Renal achieved the highest admission conversion rate (53.49%).
Patient arrivals peaked at 11 PM, indicating late-evening demand pressure.
More than 4,600 patients required admission, highlighting significant inpatient demand.
Average patient satisfaction remained at 5/10, suggesting opportunities to improve service quality.
Department-level performance analysis enables data-driven staffing and operational planning decisions.
